================================================================================
  PMC8 ESP32 Over-the-Air (OTA) Firmware Update — User Guide (V2)
================================================================================

This guide is for ota_update_v2.py — the V2 update script.

This guide explains how to update the ESP32 firmware on your PMC8 telescope
mount controller over WiFi, without opening the case or using any special
hardware. All you need is a PC, a USB cable, and Python.

The V2 script supports two ways to connect:

  DIRECT mode:  Your PC connects to the PMC8's own WiFi network.
                No home WiFi needed. Simplest method.

  LAN mode:     Your PC and the PMC8 both connect to your home WiFi.
                Needed if DIRECT mode is not available.

The script automatically detects which mode to use.


WHAT YOU NEED
-------------
- PMC8 mount controller (firmware 20A02.x or later)
- USB cable connecting the PC to the PMC8
- Python 3 installed on the PC (see "Installing Python" below)
- The following files (provided together):
    ota_update_v2.py    — the update script
    (firmware .bin)     — the new firmware (you will be asked for the filename)

For LAN mode only:
- A home WiFi network (2.4GHz — the ESP32 does not support 5GHz)
- The PC must be on the same WiFi network the PMC8 will connect to


INSTALLING PYTHON
-----------------
The script requires Python 3. If you already have it, skip ahead.

Option A — Microsoft Store (easiest):
  1. Open the Microsoft Store (Start menu -> "Microsoft Store")
  2. Search for "Python 3"
  3. Click "Get" or "Install" on the latest Python 3.x
  4. Done — Python is automatically added to PATH

Option B — python.org:
  1. Go to https://www.python.org/downloads/
  2. Download the latest Python 3 installer for Windows
  3. Run the installer
     *** IMPORTANT: CHECK THE BOX "Add python.exe to PATH" ***
     Then click "Install Now"
  4. Close and reopen your Command Prompt / PowerShell

To verify Python is installed, open a Command Prompt and run:
  python --version

You should see "Python 3.x.x".

NOTE: The script will automatically install any other required packages
(pyserial) — you do not need to install them yourself.


BEFORE YOU BEGIN
----------------
- The PMC8 must be powered on and fully booted (wait ~15 seconds after
  power-on before starting)
- Know which COM port the PMC8 is on (see "Finding Your COM Port" below)
- Close any other programs that might be using the COM port (ExploreStars,
  SkySafari, terminal programs, etc.)
- For DIRECT mode: connect your PC to the PMC8's WiFi network and
  make sure no other devices are connected to it
- For LAN mode: know your WiFi network name (SSID) and password


FINDING YOUR COM PORT
---------------------
1. Connect the USB cable from your PC to the PMC8
2. Open Device Manager:
   - Press Windows key + X, then click "Device Manager"
   - Or: Start menu -> type "Device Manager" -> Enter
3. Expand "Ports (COM & LPT)"
4. Look for your USB-serial adapter (e.g., "USB Serial Port (COM3)")
5. Note the COM port number (e.g., COM3, COM6)

If you don't see a COM port, you may need to install the driver for your
USB-serial adapter (FTDI, CH340, or CP210x — check the adapter's label).


PREPARE THE UPDATE FILES
-------------------------
Place the script and firmware file in the same folder. For example:

  C:\PMC8_Update\
    ota_update_v2.py
    esp-at.bin          (or whatever the firmware file is named)


GETTING HELP
-------------
To see all available command-line options, run:

  python ota_update_v2.py --help


RUN THE UPDATE — DIRECT MODE
------------------------------
Use this method when your PC is connected to the PMC8's WiFi network.

1. Connect your PC's WiFi to the PMC8 network (usually "PMC8_XXXX")
2. Disconnect any other devices from the PMC8 WiFi
3. Make sure the PMC8 is powered on and fully booted
4. Close any programs using the COM port
5. Open a Command Prompt / PowerShell and navigate to your folder:

     cd C:\PMC8_Update

6. Run the script (it will prompt you for COM port and firmware filename):

     python ota_update_v2.py

   Or provide everything on the command line:

     python ota_update_v2.py --port COM3 --firmware esp-at.bin

   The script will auto-detect DIRECT mode (your PC is on 192.168.47.x).

7. The script shows the installed and new firmware versions, then asks
   if you want to proceed.

8. Wait for the update to complete (typically 2-4 minutes).

9. Done! The PMC8 is already back in AP mode — ready to use.


RUN THE UPDATE — LAN MODE
---------------------------
Use this method when your PC is on a home WiFi network.

1. Make sure the PMC8 is powered on and fully booted
2. Close any programs using the COM port
3. Open a Command Prompt / PowerShell and navigate to your folder:

     cd C:\PMC8_Update

4. Run the script:

     python ota_update_v2.py

   The script will detect LAN mode and prompt you for:
     - COM port (e.g., COM3)
     - WiFi SSID (your home network name — case-sensitive!)
     - WiFi password (case-sensitive!)
     - Firmware filename (press Enter for default: esp-at.bin)

   Or provide everything on the command line:

     python ota_update_v2.py --port COM3 --ssid "YourWiFiName" --password "YourWiFiPassword" --firmware esp-at.bin

5. The script shows the installed and new firmware versions, then asks
   if you want to proceed.

6. Wait for the update to complete (typically 2-4 minutes).

7. Done! Power cycle the PMC8 to return to normal operation.

TROUBLESHOOTING
---------------

"Could not open COMx" (Phase 1):
  - Is the correct COM port specified?
  - Is another program using the COM port? Close it and try again.

"ESP32 not responding to AT commands" (Phase 2):
  - Is the PMC8 fully booted? Wait at least 15 seconds after power-on.
  - Power cycle the PMC8 and try again.

"PC is on PMC8 subnet but not at expected 192.168.47.11" (Phase 1):
  - Other devices are connected to the PMC8 WiFi.
  - Disconnect all other devices, then disconnect and reconnect your PC.

"Failed to join WiFi" (Phase 3, LAN mode only):
  - Is the SSID exactly correct? WiFi names are case-sensitive.
    Example: "NETGEAR85" is NOT the same as "netgear85"
  - Is the password correct?
  - Is the WiFi network 2.4GHz? The ESP32 cannot connect to 5GHz networks.
  - Is the PMC8 within range of the WiFi router/access point?

"Could not start OTA transfer" (Phase 5):
  - The ESP32 firmware may be too old to support OTA (it does not have the
    AT+USEROTA command). You will need to do a full serial flash using the
    factory image (factory_WROOM-32.bin). See FLASH_QUICK_START.txt.
  - The firewall may be blocking the connection. Try running the Command
    Prompt as Administrator (right-click -> "Run as administrator").
  - Check that no other program is using port 8000.

"Could not start HTTP server on port 8000" (Phase 4):
  - Another program is using port 8000. Close it and try again.

OTA transfer starts but never completes:
  - Check that the PC and PMC8 can reach each other on the network
  - Try running as Administrator for firewall access
  - Make sure the WiFi signal is strong

"PC has not reconnected to the PMC8 WiFi" (after reboot, DIRECT mode):
  - The PMC8 rebooted and your PC lost the WiFi connection.
  - Reconnect your PC to the PMC8 WiFi and press Enter.
  - If it still fails, the update may still have succeeded — the script
    will check the firmware version via serial and let you confirm.

"Firmware file is too large for OTA" (Phase 1):
  - You selected the full factory image (factory_WROOM-32.bin) instead of
    the OTA binary (esp-at.bin). The OTA binary is under 2MB. The factory
    image is a full 4MB flash and must be loaded via serial — see
    FLASH_QUICK_START.txt.

"python is not recognized":
  - Python is not installed or not on PATH. See "Installing Python" above.

If all else fails:
  - Power cycle the PMC8 — the previous firmware is still intact.
  - The OTA process is safe: if it fails, the PMC8 keeps its existing
    firmware. You can always try again.


COMMAND-LINE OPTIONS
---------------------
Run "python ota_update_v2.py --help" to see all options.

  --port PORT       COM port (e.g., COM3)
  --ssid SSID       WiFi network name (LAN mode only)
  --password PASS   WiFi password (LAN mode only)
  --firmware FILE   Firmware filename (default: esp-at.bin)

Options can be given in any order. If you omit any, the script will
prompt you to enter them.


SAFETY NOTE
-----------
The OTA update is safe. If the transfer fails or is interrupted:
  - The PMC8 keeps its existing working firmware
  - Just power cycle and try again
