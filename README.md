# ExplorestarsLite — Android

Standalone Android app for controlling Explore Scientific PMC-Eight™ telescope mounts. No PC needed.

## Download

Download `com.explorestarlite.app-Signed.apk` to your Android phone and tap to install.

## Install

1. Tap the downloaded APK file
2. Allow "Install from unknown sources" if prompted
3. If Google Play Protect warns you, tap "Install anyway"
4. Open the app, go to Setup, enter your PMC-Eight IP address, and tap Connect

## Features

- 10,900+ object catalog with altitude-based visibility coloring
- GoTo, Sync, Tracking (Sidereal/Solar/Lunar), Meridian Flip
- Spiral Search with transport controls
- Thumb pad centering with Fine/Coarse symmetric rates
- Solar system targets: planets, Moon, Sun, minor planets
- Night vision mode for dark adaptation
- GPS location, firmware config reader/writer
- Works alongside Alpaca/ASCOM and planetarium software
- Built-in user manual with QR download link
