# PMC-Eight Propeller Firmware

Latest PMC-Eight Propeller firmware binary for use with the UFCT (Universal Firmware Configuration Tool).

## Current Version

**20A02.1.8.2.bt** — Bluetooth Classic (SPP) variant

### What's in this update
- Required for ExplorestarsLite and HTTP bridge operation

## How to Update

1. Download `20A02.1.8.2.bt.binary` from this branch
2. Use the **UFCT** (Universal Firmware Configuration Tool) to flash the firmware to your PMC-Eight
3. Follow the UFCT instructions for your mount model

## Compatibility

- iExos-100
- EXOS-2
- Losmandy ES G-11
