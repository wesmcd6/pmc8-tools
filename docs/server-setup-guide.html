ExplorestarsLite Server Setup

This guide shows you how to run ExplorestarsLite on a Windows PC so you can control your PMC-Eight telescope mount from your iPhone or Android phone using a web browser.

What you will do:

  1. Install two free programs on your PC (one time only)
  2. Download ExplorestarsLite
  3. Double-click a file to start the server
  4. Open a web address on your phone
How to open a Command Prompt on Windows:
Press the Windows key on your keyboard (the key with the Windows logo, bottom-left area). Type cmd. Click on Command Prompt when it appears. A black window will open. This is where you type commands and press Enter to run them.

Step 1 — Install Two Programs (One Time Only)

You need to install two free programs. If they are already installed, you can skip to the next step.

Program 1: Node.js

Check if you already have it: In the Command Prompt, type this and press Enter:

node --version

If you see a number like v22.16.0 — you already have it. Skip to Program 2.

If you see an error message — you need to install it:

  1. Download the Node.js installer directly: Node.js v22 Windows Installer
    (Or go to nodejs.org/en/download, select Windows, and download the .msi installer)
  2. Open the downloaded .msi file and click Next / Install until it finishes (do not change any settings)
  3. Close your Command Prompt window and open a new one
  4. Type node --version again to make sure it works

Program 2: Caddy

Check if you already have it: In the Command Prompt, type this and press Enter:

C:\caddy\caddy.exe version

If you see a number like v2.10.0 — you already have it. Go to Step 2.

If you see an error message — you need to install it:

  1. Open this website: https://caddyserver.com/download
  2. Under Platform, select Windows amd64
  3. Click the blue Download button (do not add any extra plugins)
  4. A file called caddy_windows_amd64.exe (or similar) will download
  5. Open File Explorer (the yellow folder icon on your taskbar)
  6. Go to This PC → Local Disk (C:)
  7. Right-click in the empty space and choose New → Folder
  8. Name the new folder: caddy
  9. Move the downloaded file into the C:\caddy folder
  10. Rename the file to caddy (right-click → Rename). Just type caddy — do not type .exe because Windows adds that automatically. If you type caddy.exe the file may end up named caddy.exe.exe which will not work.
  11. Verify: Open a Command Prompt and type: C:\caddy\caddy.exe version You should see a version number like v2.10.0. If you see an error like "not recognized", check that the file is named exactly caddy.exe (not caddy.exe.exe) and is inside C:\caddy\.

Step 2 — Download ExplorestarsLite

  1. Open this website: https://github.com/wesmcd6/pmc8-tools/tree/pwa-server
  2. Click the green Code button
  3. Click Download ZIP
  4. When the download finishes, find the ZIP file (usually in your Downloads folder)
  5. Right-click the ZIP file and choose Extract All...
  6. Choose where to put it — for example, C:\ExplorestarsLite
  7. Click Extract
Important: After extracting, open the folder and make sure you can see start-servers.bat directly. If instead you see another folder inside called ExplorestarsLite-master, open that folder — that is where your files are. You can move them up one level or just work from that inner folder.

Step 3 — Start the Server

  1. Open File Explorer and go to the folder where you extracted ExplorestarsLite
  2. Find the file called start-servers.bat
  3. Double-click start-servers.bat

A black window will open. It will:

  • Check that all three programs are installed
  • Show your PC's IP address (a number like 192.168.0.36) — write this down
  • Start the server

When you see "All servers started!" and the two web addresses — the server is ready.

First time only — Windows Firewall: Windows may show a security popup asking if you want to allow the programs to communicate on the network. Click "Allow access" for each popup. If you click "Cancel" by mistake, your phone will not be able to connect. See the Troubleshooting section below to fix this.
Keep this window open. The server runs as long as this window is open. To stop the server, press any key in this window, or just close it.

Step 4 — Open the App on Your Phone

  1. Make sure your phone is connected to the same WiFi network as your PC
  2. Open the web browser on your phone (Safari on iPhone, Chrome on Android)
  3. In the address bar, type the address shown in the server window. It will look like: http://192.168.0.36:5257 Use the IP number from your server window — it may be different from this example.
  4. Press Go / Enter
  5. The ExplorestarsLite app should load on your phone
Type the address exactly. It starts with http:// (not https), then the IP number, then :5257 (colon, then the number 5257). No spaces.

Step 5 — Connect to Your PMC-Eight Mount

  1. Power on the PMC-Eight mount
  2. The mount and your phone must be on the same WiFi network as the PC
  3. In the app on your phone, you will see the Setup page
  4. Enter the PMC-Eight IP address:
    • If the mount is on your home WiFi: use the IP your router gave it
    • If you are not sure: ask the person who set up the mount
  5. Select your Mount Model (e.g., iExos-100, EXOS-2, or Losmandy G-11)
  6. Tap CONNECT
  7. If it works, you will see "Connected" and the firmware version number
Important: The phone, the PC, and the PMC-Eight mount must all be on the same WiFi network. If the mount is in "AP mode" (creating its own WiFi network called PMC-Eight_xxxx), the phone cannot reach both the mount and the PC at the same time. The mount needs to be connected to your home WiFi for this setup to work.

Optional — Add to iPhone Home Screen

You can make the app look and feel like a real app on your iPhone:

  1. With ExplorestarsLite open in Safari, tap the Share button (the square with an upward arrow, at the bottom of the screen)
  2. Scroll down the list and tap Add to Home Screen
  3. Tap Add in the top-right corner

Now you have an ExplorestarsLite icon on your home screen. Tap it to open the app directly.

Optional — HTTPS for Polar Alignment (iPhone Only)

The polar alignment feature needs access to the iPhone's compass. Apple requires HTTPS for this. If you do not need polar alignment, you can skip this.

  1. Install a security certificate on your iPhone — follow the separate guide: iPhone HTTPS Setup
  2. After installing the certificate, open Safari and go to: https://192.168.0.36:5256 (Use your PC's IP number, and note: https not http, and port 5256 not 5257)
  3. The app works the same way, but now the polar alignment compass works

Troubleshooting

The phone cannot load the app

  • Check WiFi: The phone and PC must be on the same WiFi network. If your phone is using mobile/cellular data, it will not work.
  • Check the address: Make sure you typed it exactly right. It must start with http:// and end with :5257
  • Check the server: Look at the black server window on the PC. Is it still running? Does it show "All servers started"?
  • Windows Firewall: This is the most common problem. Windows may have blocked the connection. To fix it:
    1. Click the Start button (Windows logo)
    2. Type Windows Security and open it
    3. Click Firewall & network protection
    4. Click Advanced settings (this opens a new window)
    5. Click Inbound Rules on the left side
    6. Click New Rule... on the right side
    7. Select Port, click Next
    8. Select TCP, type 5255, 5256, 5257 in the box, click Next
    9. Select Allow the connection, click Next
    10. Check all three boxes (Domain, Private, Public), click Next
    11. Type a name: ExplorestarsLite, click Finish
    Now try the phone again.

"node" is not recognized

  • You must close and reopen the Command Prompt after installing Node.js. The old window does not know about the new program.

The server shows an error about "address already in use"

  • Another program is using the same port number. Close any other server programs and try again. Or restart your PC.

The app loads but CONNECT fails

  • Check that the PMC-Eight is powered on
  • Check that you entered the correct PMC-Eight IP address
  • Check that the mount is on the same WiFi network

Stopping the Server

Go to the black server window on the PC and press any key. It will stop all servers and close. You can also just close the window.
